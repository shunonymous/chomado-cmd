### What's needed

C++14に対応したコンパイラ(clang推奨)、Boost、CMakeが必要です。
また、マルチバイト文字が表示可能な端末エミュレータが必要です。
環境や設定によって文字化けしますのでよろしくやってください。
(手元のlxtermはダメでした。)

### How to build

    > $ cd /path/to/dir/chomado-cmd
    > $ mkdir build
    > $ cd build
    > $ cmake ..
    > $ make
